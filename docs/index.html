Few Tips on how to write code on sequential.js

  • Use load( urlString ) to load any external JavaScript library
  • Use console.log( ... ) to display any valuable output from your code
  • Copy the URL as is, to share it or for an IFrame on your site

Current Gotchas/Issues

  • You can only use console.log once
  • Typing a for-loop might end up with executing an infinite one if you are not fast enough.for (var i=0;i<100; )
  • It's possible to write a snippet which appears to work but it's broken as it relies on a variable which you defined and then removed.
  • Your JavaScript is not fully sandboxed. Watch out not to trip on the variables used by the page
